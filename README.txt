== Description ==

This plugin is a part of and serves as ready made Astrology pages for your website.

This uses <a href="https://vedicastroapi.com/">VedicAstroAPI</a> in the backend, You get a multitude of features as mentioned.

*   Daily & Weekly prediction. - (You can choose between  - Sun sign or Moon Sign depending on your audience)
*   Kundli - Planetary details, kindly details, divisional charts, mahadashas, ashtakvarga and doshas. 
*   Panchang - Yoga, Karna,Ayanamsha, rahukaal and many more
*   Marriage Matching with Kundli
*   Planets in retrograde
*   Numerology - Destiny and Personal day prediction
*   Moon Calendar

All can be added to your website just in a couple of clicks.

== Installation ==

1. Upload the 'vedicastro' folder to the directory '/wp-content/plugins/'.
2. Activate the plugin using the 'Plugins' menu in WordPress.
3. You can adjust the necessary settings using your WordPress admin panel in 'VedicAstro' section.
